# api-node-gobarber-rocketseat: 

# API em NodeJS de uma aplicação para marcar horário na barbearia


## Tecnologias

 - NodeJS
 - Express
 - Express-async-errors
 - Express-handlebars
 - Nodemon
 - Sucrase
 - MongoDB
 - Redis
 - ESlint + Prettier
 - @sentry/node
 - bcryptjs
 - bee-queue
 - cors
 - date-fns
 - JWT
 - Multer
 - nodemailer
 - Youch
 - Yup


## Run
 - git clone https://github.com/iarlenaquiles/gobarber-api.git
 - yarn
 - yarn dev
